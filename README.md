# bdd-sample

## Laptop Setup

* Install Git https://git-scm.com/downloads
* Install Node https://nodejs.org/en/download/

Run the following in your terminal:

```sh
git clone git@github.com:BCWResearch/cucumber-example.git
npm i
npm test
```
